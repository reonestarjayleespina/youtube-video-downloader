   Terminal 1:

   └─ http://127.0.0.1:5173

📊 TECHNOLOGY STACK
═══════════════════════════════════════════════════════════════════════════════

  • Python 3.8+
  • FastAPI (modern web framework)
  • uvicorn (ASGI server)
  • yt-dlp (YouTube extraction)
  • FFmpeg (audio/video processing)
  • Pydantic (data validation)

  • Vue 3 (Composition API)
  • Tailwind CSS (utility-first styling)
  • Axios (HTTP client)
  • Vite (build tool)

COMMUNICATION:
  • REST API over HTTP
  • JSON data format
  • CORS enabled

STORAGE:
  • File system (backend/downloads/)
  • Browser LocalStorage (history)
